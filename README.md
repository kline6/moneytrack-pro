# MoneyTrack Pro（随手记账Pro）

## 概述
MoneyTrack Pro 是一个面向个人的生产级安卓记账与财务管理应用，v1 聚焦“单用户可上线”的闭环体验，后续再演进到多用户、多端同步和 AI 增强能力。

## 已确认的决策
- **币种**：人民币（CNY）
- **首版用户模型**：单用户（本地账号 + 云端存储），不做复杂多租户
- **交付策略**：先做可上线、可维护、可验证的 v1，再按阶段扩展

## v1 目标
### 必须完成
- 安卓端快速记账、交易管理、分类管理
- 首页概览：本月支出、今日支出、预算剩余、最近交易
- 月度预算与分类预算
- 月/周/分类统计与趋势分析
- 数据导出基础能力
- 统一鉴权、日志、错误处理、输入校验
- 离线记账与基础重试同步
- 生产级后端、数据库、容器化与 CI/CD 方案

### 不在 v1 范围，但保留扩展点
- 多用户/多设备复杂同步与冲突仲裁
- 多模型 AI 分析（OpenAI / Claude / Gemini / DeepSeek 等）
- 复杂年度预算、预测、报表协作
- 图片对象存储 CDN 全链路
- 企业级压测、红队与完整 E2E 自动化体系

## 为什么这样设计
1. 先保证核心账务闭环稳定
2. 避免首版模型和接口过度膨胀
3. 让数据库和 API 保持可演进，不被早期复杂需求绑死

## 后续扩展方向
- v1.x：更强筛选、导出、图表、设置能力
- v2：AI 分析、年度预算、预测、报表
- v3：多端同步、对象存储、协作、商业化增强
